## Nomenclature

| Term | Definition |
|------|------------|
| **Documentation Coverage** | Percentage of modules with corresponding documentation files |
| **Functional Documentation** | User-facing documentation explaining what a module does (in `docs/modules/`) |
| **Technical Documentation** | Developer-facing documentation explaining how a module works (in `docs/technical/`) |
| **Test Documentation** | Descriptions of test scenarios and acceptance criteria (in `docs/test/`) |
| **Drift** | Time difference (in days) between the last code change and last documentation update |
| **Staleness** | Documentation that hasn't been updated in >90 days (high drift) |
