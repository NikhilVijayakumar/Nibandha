## âš™ï¸ Configuration (Optional)

> [!NOTE]
> **Optional:** This report works without configuration, but you can customize mypy behavior.

To configure type checking strictness, add to `pyproject.toml`:

```toml
[tool.mypy]
python_version = "3.12"
warn_return_any = true
warn_unused_configs = true
disallow_untyped_defs = true

# Per-module overrides
[[tool.mypy.overrides]]
module = "tests.*"
disallow_untyped_defs = false
```

**Common options:**
- `strict = true` - Enable all strict checks
- `ignore_missing_imports = true` - Ignore untyped libraries
- `check_untyped_defs = false` - Skip checking untyped functions
