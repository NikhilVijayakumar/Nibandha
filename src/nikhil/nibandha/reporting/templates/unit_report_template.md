## Nomenclature

| Term | Definition |
|------|------------|
| **Unit Test** | An automated test that validates a single component or function in isolation |
| **Test Suite** | A collection of related unit tests |
| **Assertion** | A statement that validates expected behavior (if it fails, the test fails) |
| **Pass Rate** | Percentage of tests that completed successfully without failures |
| **Code Coverage** | Percentage of code lines executed during test runs |
| **Test Duration** | Time taken for a test or suite to complete |
| **Fixture** | Setup code or data used to prepare the testing environment |
